# Deep Research 多媒体佐证收集指南

在现代的高质量调研工作中，图片、图表、供应链地图及系统架构等多媒体佐证材料对结论的可信度至关重要。本指南规范了多媒体佐证的采集、下载过滤、本地化存储及嵌入标准。

## 1. 过滤与筛选策略 (Filtering Policy)

并非所有网页图片均具有佐证价值。子代理与 Lead Agent 在识别图片时，必须严格区分：

### 🎯 必须收集的高价值媒体：
- **数据图表**：折线图、柱状图、饼图、雷达图、数据对比表截图。
- **系统/流程图**：系统架构图、数据流图、工艺流程图、组织架构图、供应链网络图。
- **产品/UI 截图**：软件操作界面、实体产品外观、厂区或设备实地照片。
- **证明文件/公文**：官方公告批文、专利说明图纸、合同关键条款截图。

### ❌ 必须忽略的噪音媒体：
- **广告与推广**：网页横幅广告、赞助商图标。
- **网页 UI 元素**：按钮图标、面包屑导航小箭头、背景装饰大图。
- **个人头像**：作者头像、社交网络评论区用户头像。
- **不相关插图**：新闻配图中的通用风景图、无关概念示意图。

---

## 2. 子代理（Subagent）采集阶段规范

子代理在提取网页内容时，应扫描高价值图片，并在其 `task-{id}.md` 的 `## Sources` 或 `## Findings` 中，使用标准 Markdown 格式记录图片：

### 子代理图片 Markdown 标注规范
```markdown
!["[详细的图片内容描述，供后续 LLM 理解]"](original_web_image_url "图片标题")
```

### 关键要求
1. **必须包含 `alt` 信息**：`alt` 的内容必须极具描述性，能完整概括该图片承载的核心数据或事实（参见第 4 节）。
2. **必须包含 `title` 标题**：`title` 用于在最终报告中作为图表下方的文字标题。
3. **URL 必须为原始 URL**：使用网页中图片的绝对路径地址。

---

## 3. Lead Agent 下载与本地化流水线 (Localizing Pipeline)

在任务笔记收集完毕（P2 结束）与大纲及报告撰写（P4/P5 启动）之间，Lead Agent 必须执行专门的**多媒体统一下载与本地化阶段**：

1. **扫描与提取**：遍历 `findings/task-*.md`，解析出所有的图片 Markdown 节点。
2. **下载与保存**：
   - 提取原始 URL。
   - 在项目根目录的 `assets/` 下生成本地文件名，命名规则为：`assets/[Task_ID]_[Timestamp]_[Hash].[ext]`。
   - 使用带防盗链伪装的 Python 脚本或 Curl 命令将图片下载至本地（参考 `User-Agent` 与 `Referer` 头）。
3. **失败回退处理**：
   - 若某张图片下载失败（超时、403、404 等），尝试重试。
   - 重试仍失败，则回退为引用原始 Web 链接，并在日志中记录 `warning`。
4. **路径重写**：
   - 在生成 P4 大纲与 P5 草稿时，将报告中的图片链接替换为**相对于报告文档的相对路径**（例如：`assets/task-a_1716943800_sha256.png`）。
5. **看板记录**：将下载成功的资产在 `kanban/project_state.json` 的 `media_registry` 中登记。

---

## 4. LLM 友好的 Alt 标签编写规范 (Descriptive Alt Text)

为了让其他 LLM（或在无法加载图片的阅读器中）能完美理解调研报告的内容，`alt` 信息不可简写为 `alt="chart"`。必须遵循如下编写结构：

### 编写公式：
`[图表类型] + [数据实体与指标] + [趋势/核心事实结论] + [横纵轴/关键数值点]`

### 正确与错误示例对照

- ❌ **错误**：`![稀土份额](https://example.com/chart.png "全球稀土份额")`
- ❌ **错误**：`![中国稀土冶炼全球占比89%](https://example.com/chart.png "全球稀土冶炼份额")`
- ✅ **正确**：
  ```markdown
  !["全球中重稀土开采与冶炼份额占比饼图。图表显示中国在开采环节占据了全球70%的份额，而在中重稀土的冶炼和分离环节，中国的全球占比高达89%，呈现绝对垄断态势，其余11%由东南亚和澳大利亚分摊。"](assets/task-a_1716943800_sha256.png "2024年全球中重稀土冶炼份额图")
  ```

- ❌ **错误**：`![系统架构图](https://example.com/arch.png "系统架构")`
- ✅ **正确**：
  ```markdown
  !["芯片生产供应链系统架构图。该图从左到右展示了原材料输入、EDA设计、光刻制造、封装测试、到最终整机组装的五个核心阶段。其中，原材料输入端标红高亮了稀土精矿与高纯度靶材，并用双向虚线连接了出口政策准入模块。"](assets/task-b_1716943800_sha256.png "半导体精细材料供应链流向架构图")
  ```
