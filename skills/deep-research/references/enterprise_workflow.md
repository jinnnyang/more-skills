# Enterprise Research Workflow (Corporate Analysis Professional Overlay)

The Enterprise Research Mode is not a separate parallel workflow detached from the main `deep-research` pipeline. Instead, it serves as a **"professional configuration overlay"** that layers advanced corporate data criteria, quantitative frameworks, and rigorous multi-tier quality checks on top of the standard Phase 0 to Phase 7 lifecycle.

When the Enterprise Mode is activated in P0 Intake, the Lead Agent MUST seamlessly execute these specialized specifications at each corresponding phase.

---

## 🧭 Phased Lifecycle Overlay Mapping

```
[P0: Setup] ──→ Intake: Align on legal entity boundaries, depth, & peers
     │
[P1: Taskboard] ──→ Task Decomposition: Strictly map tasks to the Six Ingestion Dimensions
     │
[P2: Dispatch] ──→ Exploration: Ingest via subagent_prompt using standard methodology
     │              └── 📚 Reference: enterprise_research_methodology.md
     │
[P3: Wash] ──→ Governance: De-duplicate sources and cross-validate multi-source financial metrics
     │
[P4: Outline] ──→ Outline: Map 7-chapter outline and execute corporate quantitative frameworks
     │              └── 📚 Reference: enterprise_analysis_frameworks.md
     │
[P5: Drafting] ──→ Drafting: segmented drafting loop, enforce L1/L2 Quality Gate checks
     │              └── 📚 Reference: enterprise_quality_checklist.md
     │
[P6: Review] ──→ Review: Adversarial self-review and execute L3 Document Packaging Gate
     │
[P7: Verify] ──→ Finalize: Citations audit, README updates, and lock version tag
```

---

## E1: Intake Customization (P0 Intake Phase)
During the Phase 0 Grill-Me intake Q&A, the Lead Agent MUST prompt the user for corporate-specific parameters using a structured choice panel:
1. **Precise Legal Entity Boundary**: Clarify whether the research covers the entire parent corporation, specific subsidiaries, or joint ventures.
2. **Target In-Depth Scale**:
   - Quick Scan (3-5 pages, equivalent to *Lightweight*).
   - Standard Review (10-20 pages, equivalent to *Standard*).
   - Comprehensive Deep-Dive (20-40 pages, equivalent to *Deep*).
3. **Peer Benchmark Entities**: Select 1 to 3 direct competitors or peer entities for comparative financial/operational cross-referencing.

---

## E2: Six Ingestion Dimensions (P1 & P2 Phases)
During P1 Taskboard decomposition and P2 Dispatch execution, tasks MUST strictly map to these six corporate research dimensions. Ensure queries and subagent prompts leverage:
- **📚 Data Collection Methodology**: [enterprise_research_methodology.md](enterprise_research_methodology.md)
  - *D1: Corporate Fundamentals (Company Basics - Task A)*
  - *D2: Business Model & Product Ingestion (Core Offerings - Task B)*
  - *D3: Market Share & Competitive Positioning (Market Landscapes - Task C)*
  - *D4: Financial Performance & Operational Metrics (Earnings/Unit Economics - Task D)*
  - *D5: Recent Developments & Strategic Signals (Mergers, expansions - Task E)*
  - *D6: Sensitive/Exclusive Audit Ingests (Regulatory filings, internal reports - Task F)*

## E3: Quantitative Frameworks & Calculations (P3 & P4 Phases)
During P3 Citation Washing and P4 Outline Design, after global citation indexing, the Lead Agent **MUST execute** four quantitative modeling frameworks. The outline MUST allocate designated chapters/appendices to present these outputs:
- **📚 Quantitative Framework Guide**: [enterprise_analysis_frameworks.md](enterprise_analysis_frameworks.md)
  - *Framework 1: SWOT Matrix with Inter-Quadrant Action Items (SO/WO/ST/WT)*
  - *Framework 2: Weighted Economic Moat Rating (7 Moat vectors graded A+ to C)*
  - *Framework 3: 8-Category Corporate Risk Matrix (Probability × Impact rating)*
  - *Framework 4: Weighted Core Performance Scorecard*
