众所周知，台湾省在上世纪 70-80 年代触发了两个重要事件，一是**经济腾飞**，二是**戒严解除**。

在政治松绑和中产崛起的大背景下，各种敛财、骗色的歪门邪道开始冒头。

![这是自媒体“飞碟说”的动画插画，背景是带聚光灯效果的蓝色舞台。左侧一只黑色兔子双眼变成粉色爱心，开心抱着印美元符号的钱袋，周围堆放大量金币和成捆钞票，旁边标注文字“敛财”；右侧站着一只穿粉色连衣裙的粉色猫咪，双臂交叉抱胸、神情不屑，它腿后趴着一只带粉色爱心眼的黑兔子，旁边标注文字“骗色”；图片底部配文“各种敛财、骗色的歪门邪道开始冒头”，左上角是“飞碟说”的品牌标识。](./【飞碟说】浅谈台湾邪教_assets/讽刺敛财骗色歪门邪道的动画插画.webp "讽刺敛财骗色歪门邪道的动画插画")

其中比较出圈的邪教头子有“显相学会”的**宋七力**、“观音法门”的**张兰君**、自称禅宗第八十五代宗师的**黄明亮**、“紫衣神教”的**刘锦隆**，以及华兴灵修中心的**徐浩城**。

这些怪东西的共同之处，是通过朋克人设和抽象话术，一步步筛选 TA 打造饭圈，最后**吸血牟利**。

人气、财力达到一定程度的头子，如黄明亮，甚至野心爆棚，短暂**成立过政治党派**。

他们中最抽象的人是**徐浩城**。

![这是飞碟说的动画截图，左上角是带播放图标的橙色“飞碟说”品牌标识；画面中心是一名戴浅框眼镜、留短黑发的中年男子，他身穿胸口印有汉字“邪”的黄色宽袖长袍，张开双臂站在放射状暖黄色光束背景前；画面下方是一群仅露出头部背影、颜色包含绿、蓝、黑的听众人群剪影；图片底部配有白色字幕：“他们中最抽象的人是徐浩城”](./【飞碟说】浅谈台湾邪教_assets/飞碟说动画谈论徐浩城的场景.webp "飞碟说动画谈论徐浩城的场景")

这个花名徐少龙的奸奇大魔处心积虑搞邪教，很大程度上只为圆自己的偶像大师梦。

在用“迷弟迷妹”的钱发专辑、开只卖自己唱片的连锁店之余，他还宣扬俗套的**地球末日论**，向信徒们收了一大笔流浪太空的船票钱。

![这是飞碟说的动画插图，深酒红色太空背景上点缀着多颗四角星形的白色星星，画面中心是一枚向右上方飞行的卡通火箭：火箭主体为白色，带有红色装饰条纹、蓝色尾翼，侧面排列着一排黄色圆点，头部有蓝色小圆顶，圆形驾驶舱内有一名男性乘员，尾部喷射出橙黄色火焰；左上角带有橙色播放样式的飞碟说品牌标识，图片底部配有白色中文字幕：向信徒们收了一大笔流浪太空的船票钱](./【飞碟说】浅谈台湾邪教_assets/飞碟说动画中的卡通太空火箭插图.webp "飞碟说动画中的卡通太空火箭插图")

东窗事发前，徐浩成整整忽悠了**14000 余人**入教，期间搜刮近**8000 万新台币**，纳 200 多名女信徒为后宫。

而他被捕的原因，却跟宣扬自己的歌声能治新冠有关。甚至到最后，这个老神棍都没受到严厉的制裁。
