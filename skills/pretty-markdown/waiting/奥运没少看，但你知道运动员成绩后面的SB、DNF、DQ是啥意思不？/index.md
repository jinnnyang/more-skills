******视频文案******

我有一个智障表弟，奥运比赛没少看，运动员成绩后面的英文缩写却一个都不认识。

咱就从最常见的那几个说起。

![这是一幅扁平风格动画插画，左上角带有“飞碟说”的品牌logo与文字；背景为浅蓝色天空，飘着两朵浅白云朵，远处有一排绿色灌木丛，下方是红色塑胶田径跑道，跑道旁有绿地与白色护栏；五只不同造型、不同颜色的卡通动物都朝向画面右侧在跑道上跑步，从左到右（从后到前）依次为粉色小猫、黑色兔子、蓝色猫咪、浅黄小熊、红色狐狸。](https://mmbiz.qpic.cn/mmbiz_png/DlicUgzw0fWqgu6eBYcuy5iaicRle2fliaR6BKq2HBxAWV96L6pZkvhIET2h7iaBpK8N0YTeWIu8lqP1DhWSh4ILhew/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1 "卡通动物田径场赛跑")

OR 是 Olympic Record 的缩写，意思是奥运会纪录。

WR 则是 World Record 的缩写，表示世界纪录。

SB 不是国粹，而是 Season Best 的缩写，即个人赛季最好成绩。

DQ 跟冰淇淋没关系，它是 Disqualified 的缩写，说明运动员未取得晋级下一轮的资格。

![这是飞碟说的动画插画，左上角是带有播放图标的橙色飞碟logo，下方标注文字“飞碟说”；背景为浅蓝色天空，天空下方排列着深浅不一的绿色灌木丛，灌木丛前方依次是灰色围栏、绿色草地，最前方是红棕色的赛跑跑道；跑道上有一只头部四肢为黑色、龟壳为绿色的卡通乌龟，乌龟身上斜盖着一个红框红字的印章，印章内写有大字“淘汰”](https://mmbiz.qpic.cn/mmbiz_png/DlicUgzw0fWqgu6eBYcuy5iaicRle2fliaR6UbFdQ1OtQzs6aGTzibdIvz2PAGVXQ2ldIt7iaIKauQicP8tib39ujh0lHw/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=2 "赛道上被标记淘汰的乌龟")

Q 刚好相反，它是 Qualified 的缩写，表示运动员取得了晋级下一轮的资格。

如果成绩栏上标注 DNS，那意味着运动员 Did Not Start，也就是未出赛。

![这是飞碟说出品的卡通插画，左上角带有橙色飞碟造型的播放图标与「飞碟说」文字标识；背景是浅蓝色天空，飘着两朵淡蓝色云朵，天空下方依次排列着绿色灌木丛、白色围栏、绿色草地，前景是带有白色分道线的棕红色塑胶跑道；五只卡通动物从左到右排列，正一同朝右在跑道上奔跑，依次为粉色小猫、黑色长耳兔、蓝色小猫、浅黄小熊、红色狐狸。](https://mmbiz.qpic.cn/mmbiz_png/DlicUgzw0fWqgu6eBYcuy5iaicRle2fliaR6BKq2HBxAWV96L6pZkvhIET2h7iaBpK8N0YTeWIu8lqP1DhWSh4ILhew/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=3 "卡通动物跑道赛跑插画")

标 DNF 则代表运动员 Did Not Finish，即未完赛。

除了以上 7 个英文缩写，大型体育赛事还经常出现 MR、CR、AR、NR、PB、WL，它们分别指代赛会记录、锦标赛记录、洲际记录、国家记录、个人最好成绩、当年世界最好成绩。
