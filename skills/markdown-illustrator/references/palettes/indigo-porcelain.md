# indigo-porcelain

Calm indigo ink on pale porcelain gray for technology, research, and analysis

## Background

- Color: Pale Porcelain Gray (#F2F4F5)
- Texture: Subtle cool paper grain

## Colors

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Background | Pale Porcelain | #F2F4F5 | Primary background canvas |
| Primary | Deep Indigo | #0A1F3D | Main text, lines, display headers, structures |
| Muted | Slate Gray | #5F6D78 | Secondary text, annotations, minor labels |
| Accent | Porcelain Blue | #315D93 | Key semantic markers, numbers, active borders |
| Soft Fill | Soft Indigo | #E5EBEF | Card backgrounds, section highlights, fills |

## Accent

Porcelain Blue (#315D93) for key semantic highlights. Use to highlight tech terms, research data points, or active borders.

## Semantic Constraint

Calm indigo ink on pale porcelain base. Restrained color usage — keep to slate gray and soft porcelain fills. Accent porcelain blue sparingly for key emphasis only. Do NOT render color names, hex codes, or role labels as visible text in the image.

## Compatible With

- `editorial-magazine` (primary, default pairing)
- `minimal` (cool variation)

## Not Recommended With

- `swiss-international` (requires clean off-white backgrounds and high-saturation single accent colors)
- `neon` (dark theme only)

## Best For

Tech manifestos, AI research summaries, data analysis, calm academic essays, and analytical writing
