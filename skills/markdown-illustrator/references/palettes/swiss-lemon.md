# swiss-lemon

Vibrant Lemon Yellow accent on off-white and charcoal for young, consumer, and retail content

## Background

- Color: Matte Off-White (#FAFAF8)
- Texture: Clean off-white matte texture, no paper grain

## Colors

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Background | Matte Off-White | #FAFAF8 | Primary background canvas |
| Primary Text | Clean Black | #0A0A0A | Main text, outlines, grid labels |
| Accent | Lemon Yellow | #FFD500 | Key semantic highlight, focus card background |
| Accent Text | Clean Black | #0A0A0A | Text rendered on top of Accent card fill |
| Muted | Grid Gray | #737373 | Minor labels, annotations, captions |
| Soft Fill | Light Gray | #F0F0EE | Neutral cards, data bars, structures |
| Hairline | Border Gray | #D4D4D2 | Grid lines, cell dividers |

## Accent

Lemon Yellow (#FFD500) is the singular high-saturation accent color. Use only for primary focus areas: a key metric, an action card, or a critical flow node.

## Semantic Constraint

Strict single-accent rules. Because Lemon Yellow is light, text on it MUST use Clean Black (#0A0A0A) for readability. No gradients or drop shadows. Do NOT render color names, hex codes, or role labels as visible text in the image.

## Compatible With

- `swiss-international` (primary, default pairing)
- `playful`

## Not Recommended With

- `editorial-magazine` (conflicts with serif print and warm paper aesthetics)
- `vintage` / `retro` / `watercolor` (requires textured canvases and softer color transitions)

## Best For

Consumer retail products, active lifestyles, fitness summaries, sporty data cards, and young/playful summaries
