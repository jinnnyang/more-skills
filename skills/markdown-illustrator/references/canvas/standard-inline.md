# standard-inline

Standard inline article illustration canvas — the default 16:9 landscape format for embedded content images.

## Dimension

- Width: 900px
- Height: 500px
- Ratio: 16:9
- viewBox: 0 0 900 500

## Style Constraints

- Min readable font-size 24px for body, 18px for captions
- Content must not bleed to edges
- Recommended padding: 40px on all sides
- For WeChat cover (900×383), core subject must stay within central 383×383 safety zone
- All inline illustrations within a single article must use the same ratio (no mixing landscape/portrait)

## Best For

WeChat article inline illustrations, blog post images, technical documentation figures, general-purpose landscape format content images
