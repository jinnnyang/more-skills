# wechat-cover

WeChat Official Account cover pair viewports specification (21:9 primary and 1:1 square sharing thumbnail)

## Dimension (Primary Cover)

- Width: 2100px
- Height: 900px
- Ratio: 21:9 (7:3)
- viewBox: 0 0 2100 900

## Dimension (Square Thumbnail)

- Width: 1080px
- Height: 1080px
- Ratio: 1:1
- viewBox: 0 0 1080 1080

## Style Constraints

- **21:9 Primary Cover**: Keeps full title, subtitle, and primary image/visual elements. Recommended padding: 80px (top/bottom) by 120px (left/right).
- **1:1 Share Card**: Uses a simplified, compressed short title derived from the main title (recommended 6-8 Chinese characters). Displays big typography, clean backgrounds, and drops cramped subtitles. Recommended padding: 80px on all sides.

## Best For

WeChat Official Account top-page main banner (21:9) and moments sharing card / chat thumbnail (1:1)
