# rednote-cover

Xiaohongshu 3:4 note cover and content card viewport specification

## Dimension

- Width: 1080px
- Height: 1440px
- Ratio: 3:4
- viewBox: 0 0 1080 1440

## Style Constraints

- Min readable font-size: 26px for body/paragraphs, 22px for labels/captions
- Title maximum lines: 2 lines (capped to prevent overflow)
- Title size recommendation: 80px - 96px
- Padding recommendation: 96px on all sides (keeps safe margin from mobile interface overlays)

## Best For

Xiaohongshu/Rednote note covers, carousel card decks, split-page tutorials, and vertical social cards
