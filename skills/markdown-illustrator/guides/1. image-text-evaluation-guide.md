# 图文配合评价指南 (Image-Text Collaboration Evaluation Guide)

本指南是一套高内聚的系统化框架，包含图文并茂的核心原则、量化技术指标、评分量表、触发判定表以及基准测试文本。用于直接评估人工编辑或 AI Agent 对 Markdown 文章的排版配图质量。

---

## 📖 一、 核心指导原则 (Foundational Principles)

1.  **叙事相得益彰**：图片用于直观梳理流程、表现变量关系和烘托情感基调；文字则用于反映复杂的思想深度，二者衔接自然，避免“为了画图而画图”。
2.  **内容为本**：配图形式必须为实质内容服务。图文报道要紧扣时代脉搏，通过精心策划实现深度挖掘，绝不以空洞的形式应付读者。
3.  **组图讲故事**：
    *   **完整性**：必须有一张大景别（远景/全景）的**主图**用于点题，副图（中景/特写）用于诠释细节，横竖构图搭配，影调对比反差合理。
    *   **逻辑性**：图片之间需建立因果或并列关系（如空间位移、工序步骤、时间节点）。
4.  **文字画龙点睛**：文字简洁准确，避免在图片说明中重复画面一目了然的常识。长文强制采用小章节/分段式框架。

---

## 📐 二、 物理与技术规范 (Quantitative Specifications)

在评估配图质量时，必须强制核对以下指标：

| 维度 | 指标项 | 规范要求 |
| :--- | :--- | :--- |
| **配图数量** | **标准数量** | 约 $2000$ 字的常规推文配置 **$4 \sim 5$ 张**正文内插图（极简短篇 3 张，步骤教程 6-8 张）。 |
| | **配图节奏** | 遵循“三段一图”，行文约每 3 个小段落（手机屏约 1~2 屏）插入一张图。 |
| | **总大小控制** | 整篇文档图片总容量控制在 **$5\text{MB}$** 以内，保障网页快速加载。 |
| **类型占比** | **修饰图 ($50\%\sim60\%$)** | 用于烘托氛围。观点类文章用偏抽象的插画，避免使用具象照片造成理解偏差。 |
| | **说明图 ($20\%\sim30\%$)** | 思维导图、饼图、折线图等，精确传达分类、占比和趋势。 |
| | **情绪图 ($20\%\sim30\%$)** | 原创照片、高清摄影或统一流派插画。**严禁使用带水印图或网络表情包**。 |
| | **动效图 ($\le 1$ 张)** | 仅限 GIF 格式，且文件体积必须 $< 1\text{MB}$。 |
| **尺寸比例** | **封面头图** | **$900 \times 383$ 像素**（比例 $2.35:1$）。下方不堆砌文字，核心元素位于中央 $383 \times 383$ 方形安全区。挂载在 YAML Frontmatter 或 H1 首行下方。 |
| | **正文内插图** | 统一为 **$900 \times 500$ 像素**（比例 $16:9$ 横版）或统一宽度为 $900\text{px}$ 的长图。禁止横竖混用。 |
| **品牌感** | **固定色系** | 整篇文章（含文字、背景和插图）的颜色总数**不超过 3 种**。<br>- 科技类：深蓝（主色）+ 明黄（点缀）+ 浅灰/白色（背景）<br>- 财经类：深灰/黑色（主色）+ 金色（辅助色）+ 浅色背景<br>- 生活类：低饱和度浅绿、淡粉、米白色 |
| | **延续风格** | 整篇文章的插图必须保持一致的视觉流派（例如全篇扁平商务风或全篇治愈手绘）。 |

## 📋 三、 配图体检与评分记录卡 (Evals Scorecard)

对文章配图成果进行人工或自动扫描，满分 5 分：

*   **5 分 (卓越)**：完全符合数量、景别逻辑、尺寸一致性、三色原则与风格延续性；图注管用简洁；**图片信息与文字具备深度互补性（带入辅助理解的第二维度信息）**；带有完整的 MITTS 透传块。
*   **4 分 (良好)**：数量与尺寸完全达标，配色与风格基本一致，叙事契合；仅在局部有微小的排版留白或图注赘余等非严重瑕疵。
*   **3 分 (合格)**：插图均有文字呼应，大体结构明晰；但封面与内文风格略有偏离（如 Notion 风混搭了极简设计插画）。
*   **2 分 (不合格)**：出现图片比例不一（如横竖交替）；或 GIF 体积 $>1\text{MB}$；或插图中含有 AI 乱码文字；或图片与正文脱离；**或说明图与正文内容纯粹机械重复（属于无效废图）**。
*   **1 分 (严重故障)**：物理图片文件丢失（File Not Found），或者图片严重喧宾夺主导致正文排版发生物理性错乱，或涉嫌数据造假/侵权水印。

### 评分卡核对表
```
[ ] 头图尺寸为 900x383 像素                    [ ] 插图比例一致（统一为 16:9 横图或宽 900px）
[ ] 头图核心元素位于中央 383x383 方形安全区       [ ] 正文插图没有横竖图混杂
[ ] 头图无底部文字堆砌，挂载在 H1 下或 yaml 中    [ ] 单张 GIF 体积 < 1MB 且全篇最多 1 张
[ ] 配图总文件容量控制在 10MB 以内               [ ] 颜色总数限制在 3 色内，且符合风格延续性
[ ] 说明图、修饰图、故事图占比符合 631 规划      [ ] 生成了可供 LLM 读屏的 MITTS 文本透传块
[ ] 说明图承载信息与正文互补，非机械重复废图
```

## 📄 四、 附录：基准测试文本 (Benchmark Cases)

### 案例 1：大众传媒图文报道类 (news-report.md)
```markdown
# 随着社会与科技的发展，“读图”成为人们热衷的一种获取信息的途径

随着社会与科技的发展，“读图”成为人们热衷的一种获取信息的途径。从某种程度上来说，信息社会已经步入了“读图时代”。图片在各类刊物上的使用量越来越大，网络、微信公众号等新媒体更是以“图配文”的形式引起读者的兴趣。我们姑且将这种有图有文字的报道形式称为图文报道。在图文报道中，图片的重要性无可厚非，但文字能够反映较复杂的思想内容，易于培养人的思维深度和多元想象力，能传递更多的纵向叙述性信息。在“读图时代”如何做好图文报道，使得新闻报道“图文并茂”，值得新闻工作者好好思考。

## 一、做到“图文并茂”的三个关键
“图文并茂”表现在编辑的观念形态上，就是用图片来配合和烘托版面上的文字，用图片来调节和美化版面。具体而言，要正确处理好图片与文字之间的关系。

### 1. 用组图来讲故事
在一篇完整的图文报道里，图片应该满足以下三个方面的特征：
*   **一是讲求完整性**。图文报道，讲求图片成组。在一组图片中，至少应该有一张点题照片，即主图。这张图片是核心，通常选取最具有视觉冲击力、最吸引人眼球、最能引发思考、最能揭示图片故事内涵的图片，起到画龙点睛的作用。而其他的图片即副图，是用来诠释细节、对主图进行补充 and 说明的。从景别上看，一般而言，主图讲求场面大气、画面完整，常常用到远景、全景，而副图一般是相对小的景别，可以是交代人物关系的中景，也可以是拍摄对象的局部、展现细部的特写。在版式的呈现上，一组图片的构图要有横竖的合理搭配，在画面色彩和影调的处理上要讲求对比与反差。
*   **二是符合逻辑性**。成组的图片之间要有一定的结构方式和内在逻辑，而不是几张图片简单、随意的叠加。图片之间可以建立因果、并列等关系，各种逻辑关系可以构成一定的情节起伏、发展和呼应。比如在游记类的图文报道中，图片可以按空间位移展开；在介绍手工艺制作的时候，图片一般以制作流程的顺序进行合理铺成；而在讲到民俗节日庆典时，图片则可以按时间节点来排列。通过合理的逻辑关系，图片可以帮助读者迅速理清人与人、人与环境的关系，读者在读图的过程中，能将图片与图片之间形成联系，通过这些联系，可以从多角度、多维度、立体地获得一个完整的信息，理解其深层次的涵义。
*   **三是实现故事性**。图文报道的图片不能仅仅追求构图美观、色彩绚烂，每一张照片都要紧扣主题。要让读者能在单张照片里读出一个“小故事”，在一组照片里读到数个“小故事”，并建立联系，形成一个“大故事”框架。事实上，在图文报道中，完成了组图的完整性和逻辑性，就基本上实现了故事性的大部分工程。图文报道要按篇幅所需进行完善，做到有头有尾，有情节、有起承转合，那样，就基本符合一个“故事”的构架了。

### 2. 用文字来画龙点睛
优美流畅的文字，可以促进读者产生发散思维，经过联想，在脑海里产生文字所描述的丰富画面，这种通过文字联想二次加工产生的画面，可以与图片呈现的实际画面进行比对，这有助于帮助读者加深对原有画面的认识。
需要注意的是，虽然我们主张图文报道的文字要优美流畅，但文字的简洁和准确是首要前提。在文字主体传达了足够清楚明了的信息之外，我们可以对文字进行适当的修饰，且要遵循“适可而止”和“文对题”的原则，同时尽量少加入主观评价。应该适度、合理地运用一些创作手段。比如，文章开头寥寥几句散文式的描写，便可以自然地将读者带入情景；运用剧本创作里常用的场景描述开头，既渲染气氛，又生动活泼……这些创作形式往往都能对通篇文字起到画龙点睛的作用。此外，文字优美还在于体例，如果是长篇幅的报道，可以采用分段式、小章节的形式，令文章框架明晰，简单易读。

### 3. 发挥好图片说明的作用
图片说明是正文本身的一个延伸阅读，许多在正文里不好交代的东西，可以放在图片说明里。图片说明指向明确，更便于读者理解。图片说明贵在简洁，它对图片起着介绍情况、说明背景、深化主题、烘托气氛等作用。值得注意的是，图片说明是为了弥补形象叙述的局限与不足，而不是重复照片已经表达清楚的内容。因此，通过图片一眼就能读懂的常识性的内容，应避免在图片说明里重复。

## 二、“图文并茂”是手段，内容才是根本
“图文并茂”终究是新闻报道的一种形式，是一种手段。如果新闻报道没有实质性的内容，不能传递给读者有用的信息，不能引发读者进行思考，形式再花俏、再有趣，也不能从根本上抓住读者。在图文报道中，只有把握时代脉搏，抓住社会热点，才能获得读者的认可。
对于民族新闻媒体来说，民族新闻从业者既是新闻工作者，也是民族工作者；既要认识到民族新闻媒体与众多新闻媒体一样存在的共性，更要认识到民族新闻工作者在社会主义精神文明建设中所肩负的重任。这对于挖掘新闻内容、并以“图文并茂”的形式呈现其内容尤为重要。在宣传报道民族工作时，要牢牢把握大局、紧扣时代脉搏，深入宣传中华民族伟大复兴的“中国梦”，深入宣传社会主义核心价值观，特别要深入宣传各民族团结友爱的优良传统和民族关系的发展主流，深入宣传少数民族和民族地区科学发展的经验成就，深入宣传少数民族对中华民族发展进步作出的巨大贡献，深入宣传民族团结进步模范典型的先进事迹。明确了这些内容，通过精心策划、挖掘好的新闻题材，将新闻报道以“图文并茂”的形式呈现在读者面前，实现对新闻资源的特色加工、深度挖掘，就能在数字化时代把握主动权，就能更好地满足少数民族群众的需求，更好地服务于民族工作。
```

### 案例 2：学术与硬核技术图表类 (academic-charts.md)
```markdown
# 高质量的论文中往往包括大量的数据和信息，如何清晰、简明地阐述这些内容就显得尤为重要

一篇高质量的论文中往往包括大量的数据和信息，如何清晰、简明地阐述这些内容就显得尤为重要，而论文的可视化是其中的关键一步。高质量的科研图表不仅能有效传达出作者想要阐述的观点，还能令审稿人与读者眼前一亮，提升论文的可读性和传播量。

然而，用对和用好科研图表并非易事，制作图表也需要掌握一定技巧，如各类图表的适用场景、绘制图表的规范、常见误区等。本期，我们将深入探讨绘制科研图表的关键要点，配合丰富的案例，详细阐述图与表的应用场景以及绘制图表的要点。

## 图与表的使用场景
图与表有各自适用的场景，用对了才能和文字相得益彰，并达到预期的效果。
*   **图片**：适合用来梳理流程、表现变量间的关系。如以下示例1描绘了流浪汉迁移方向和当地人均收入水平之间的关联。
*   **表格**：适合用来整合庞杂信息、简明地展示数据、放大对比细节。此外，表格还可以将大量信息提炼成简洁易读的形式，从而避免因文字冗长而影响读者的阅读体验。

## 五大论文图片类型及使用场景
以下是五种常见的论文图片类型及对应的使用场景。研究者可以结合制图目的和数据的具体情况酌情采用：
1.  **折线图**：用于展示一段时间内趋势或关联。
2.  **柱状图**：用于强调对比、突出分类。
3.  **饼图**：强调比例分布。
4.  **实际影像资料**：用于展示实验证据。这类图片包括：显微镜观察结果、扫描图像、或相机拍摄的动植物照片等。
5.  **示意图或模型图**：用于展示生化途径、设备结构、实验流程等内容。

## 绘图与制表的要点
想要掌握高质量绘图制表技巧，请收下以下锦囊：

### 要素齐整、注释清晰
以折线图为例，图题、图体、坐标、图例、图注等都是绘图的基础元素。这些要素如何设置、文字说明是否清楚，决定着图片是否易读、易懂。必要时也可以增加脚注或注释进行补充说明。
各个元素已在下图中用蓝色方框逐一标出，供对照参考。

### 内容一致 避免冗杂
论文中所有的图表应在字体、注释、配色、矢量图风格等方面保持统一。同时，图表中内容应经过取舍，避免内容过多、过杂，影响观感。

### 提升读者阅读体验
仔细斟酌行列标签，选用合适的字体、字号，让表格可读、易懂；在配色上注意不同类目配色的区分度，便于读者准确辨别。

除了以上锦囊之外，以下几个要点也同样不容忽视：
*   **确保数据准确**：确保图表中的数据准确且可复现。
*   **学会分图**：把过多信息塞入单个图表，不仅增加排版难度，还会让图表晦涩读懂。这种情况下，建议研究者根据情况拆分信息，绘制多个图表。
*   **考虑文章节奏**：配合文章结构适时地插入图表，切忌贪多。
*   **遵守科研道德**：避免伪造或篡改实验数据，准确、真实是重中之重。

## 图文相得益彰 表达效果加倍
绘制论文图表最忌讳“为了画图而画图”。这会导致图表和正文缺乏关联，甚至出现割裂感。以下是能够让图文相得益彰、紧密结合的小技巧：

### 文内呼应、自然衔接
不同期刊对图表位置的要求不尽相同，建议投稿前先查阅目标期刊的作者指南。此外，作者在行文中应适时交代、引出相关图表，不要让图表脱离文字介绍独立存在于论文中。

### 背景及说明信息
作者应明确且具体地说明通过图表所要陈述的观点，避免表达模糊不清。在以下两个对比例句中，第一个例句表达含糊。而第二个例句则具体说明了温度是怎么变化的、程度如何，与后文衔接更自然。
*   **不建议的写法**：`Figure 3 shows the change in temperature with time. （图3显示了温度随时间的变化。）`
*   **推荐的写法**：`The temperature fluctuation over time follows a sine-wave pattern peaking at 30 degrees (Figure 3). （温度随时间推移呈正弦波模式，峰值为 30 度。）`
```
