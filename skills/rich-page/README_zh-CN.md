# Rich Page (深度写作与版面编排) 写作技能

`rich-page` 是一个专门用于辅助智能体创作、撰写和排版高质量文章、深度报告、博客帖子以及微信公众号图文的协同写作工作流技能。它集成了先进的版面排版设计、文字转图片提示词生成，以及结构化内容配比。

---

## 核心功能

1.  **经典内容配比（631 与 361 规则）**:
    -   **631 轨道（知识/职场/成长/科技类）**：60% 干货内容 + 30% 钩子/情绪 + 10% 转化引导。
    -   **361 轨道（情感/生活/观点/故事类）**：30% 干货内容 + 60% 故事/情绪 + 10% 转化引导。
2.  **互动式大纲与配图节奏（三段一图）**:
    -   遵循“三段一图”原则（约每 3 个小段落或手机屏 1-2 屏插入一张图），提供视觉呼吸感，降低跳出率。
    -   确定统一的视觉风格、固定色系（总数不超过 3 种）与图片尺寸规范（正文 16:9，头图 2.35:1，保留中央安全区）。
3.  **分块分段撰写（Chunk-Based Drafting）**:
    -   按章节或大纲步骤一步步细化生成，避免单次生成长文本时的内容稀释与逻辑偏离，确保细节的扎实度。
4.  **连接处微调（Prose Micro-Tuning）**:
    -   在最后合并文本阶段，仔细审查段落连接处的过渡，重新润色句子以确保逻辑顺畅、语气一致。
5.  **自动化指标检测**:
    -   随包附带 `verify.js` 脚本，可快速分析文章的总字数、配图占比、Alt 提示词详细度及干货/情绪/转化比例是否达标。

## 配图与 Alt 提示词规范
当为新图片规划占位符时，`alt` 标签中必须写入高度详细的场景白描（包含构图、相机视角、主体、画风、色彩、光照），使其能直接作为 Midjourney、DALL-E 3 等生图工具的 Prompt 使用。
-   **文章头图**：尺寸 900x383px。关键视觉元素必须在中央 383x383px 安全区内，上方避免堆砌繁杂文字。
-   **正文插图**：尺寸 900x500px（16:9）。所有插图长宽比例需保持严格统一，防止阅读参差感。

## 与其他技能的协同调用逻辑

`rich-page` 不是孤立运作的，它处于一个完整的写作、插图、排版、发布生态中。以下是它与相关技能之间的调用和数据流转逻辑：

```mermaid
graph TD
    A[用户原始输入 / 主题想法 / 调研报告] --> B[rich-page (核心写作智能体)]
    
    %% 前期准备
    SD[style-distiller] -.->|Phase 1: 提炼目标文风模板| S[references/styles/]
    S --> B
    
    DR[deep-research] <-->|Phase 1: 补充事实背景与轻度调研| B
    
    %% 核心流程
    B -->|Phase 2: 规划段落大纲与配图节奏| C[大纲草稿与图片占位符]
    
    %% 插图生成
    C --> DI[doc-illustrator]
    DI -->|基于 Alt 中的场景白描生成配图| B
    
    %% 排版优化
    B -->|Phase 5: 原始 Markdown 输出| PM[pretty-markdown]
    PM -->|CJK间距美化、排版纠错与可访问性处理| D[精美排版 Markdown 文档]
    
    %% 微信发布
    D --> PW[post-to-wechat]
    PW -->|发布到微信公众号平台| E((微信公众平台))
```

1.  **`style-distiller` (文风蒸馏器)**：在写作前启动，分析爆款文章的表象与隐变量，输出风格配置文件（如 `feidieshuo-style.md`）存入 `references/styles/` 目录。如果在该目录下存在多个风格配置文件，`rich-page` 会根据输入主题推荐最佳风格，并主动询问用户的最终选择。
2.  **`deep-research` (深度调研)**：当用户输入的原始想法或数据报告内容不够详实、存在事实缺失时，在 Phase 1 阶段调用以补充背景知识。
3.  **`doc-illustrator` (配图插画师)**：在 Phase 2 或 Phase 3 阶段，当 `rich-page` 规划好配图节奏并在 `alt` 中写好详细的生图 Prompt 后，调用它来执行实际的图片生成或插画绘制。
4.  **`pretty-markdown` (排版美化)**：撰写完成后，在 Phase 5 阶段调用，用于自动为中英文/数字间添加空格，转换排版符号，提升排版美感与可访问性。
5.  **`post-to-wechat` (微信发布)**：将最后精美排版并内嵌好本地/网络图片资源的 Markdown 文章直接推送发布到微信公众号平台。
