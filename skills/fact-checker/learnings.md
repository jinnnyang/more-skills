## Known Environment Issues
- [ ] Direct download URLs from source repos are occasionally blocked by local network proxies.
- [ ] Reading large PDF source documents can cause memory/token constraints.

## Success Patterns
- [ ] Prioritize official documentation (e.g. anthropic.com/news, platform.openai.com/docs) over third-party blog posts.
- [ ] Clearly label verified claims with temporal contexts such as "As of YYYY-MM-DD".
- [ ] Organize corrections in a markdown comparison table containing source URLs.

## Failures & Anti-patterns
- [ ] Avoid making corrections to subjective opinions, tutorial instructions, or explanatory prose.
- [ ] Do not apply corrections to files without explicit user approval.
- [ ] Avoid using vague temporal references such as "latest version" or "current model".
